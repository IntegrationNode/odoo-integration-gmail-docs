Política de Privacidad

Última actualización: October 22, 2025

Resumen

Integración de Odoo para Gmail ("la Extensión") está comprometida con la protección de tu privacidad. Esta política de privacidad explica cómo la Extensión recopila, usa y protege tu información.

Desarrollador: OdooIntegrations Team

Información que Recopilamos

Datos Accedidos desde Gmail

La Extensión accede a la siguiente información de Gmail cuando abres explícitamente un email:

  • Direcciones de email de remitente y destinatario: Usadas para buscar contactos coincidentes en tu instancia de Odoo
  • Asunto y cuerpo del email: Usados cuando eliges crear oportunidades, tareas o tickets en Odoo
  • Fecha y metadatos del email: Usados para registrar emails en registros de Odoo

Datos Accedidos desde Odoo

La Extensión recupera la siguiente información de tu instancia de Odoo:

  • Información de contacto (nombres, emails, teléfonos, empresas)
  • Oportunidades CRM asociadas a contactos
  • Tareas de proyecto asociadas a contactos
  • Tickets de helpdesk asociados a contactos
  • Información de empresa y contactos relacionados

Datos de Configuración

La Extensión almacena la configuración de tu instancia de Odoo localmente:

  • URL de la instancia de Odoo
  • Nombre de la base de datos
  • Credenciales de autenticación
  • Preferencias de usuario (configuración de secciones visibles)

Cómo Usamos tu Información

La Extensión usa tu información únicamente para los siguientes propósitos:

  • Mostrar Información de Contacto: Mostrar detalles relevantes de contactos de Odoo en la barra lateral de Gmail
  • Crear Registros en Odoo: Cuando haces clic en botones "Crear", el contenido del email se envía a tu instancia de Odoo para crear oportunidades, tareas o tickets
  • Registrar Emails: Cuando haces clic en botones "Registrar Email", el contenido del email se guarda en el chatter de tu instancia de Odoo
  • Buscar y Filtrar: Buscar en oportunidades, tareas y tickets asociados a contactos
  • Caché: Almacenar temporalmente respuestas de API de Odoo localmente para mejorar el rendimiento

Almacenamiento y Seguridad de Datos

Solo Almacenamiento Local: Todos los datos se almacenan localmente en tu dispositivo usando las APIs de almacenamiento de Chrome. Ningún dato se transmite a servidores de terceros excepto tu propia instancia de Odoo.

Transmisión de Datos: La Extensión se comunica directamente con TU instancia de Odoo usando HTTPS. Tus credenciales y datos de Odoo nunca se envían al desarrollador de la extensión ni a terceros.

Gestión de Caché: Los datos en caché se borran automáticamente después de 5 minutos o cuando haces clic en el botón de actualizar. También puedes borrar todos los datos en caché eliminando la extensión.

Servicios de Terceros

ExtensionPay (Procesamiento de Pagos)

La Extensión usa ExtensionPay (https://extensionpay.com/) para la gestión de suscripciones y procesamiento de pagos. Cuando te suscribes al plan PLUS, ExtensionPay recopila:

  • Tu dirección de email para gestión de cuenta y recibos
  • Información de pago (procesada de forma segura por Stripe, no almacenada por la Extensión)
  • Estado de suscripción e información de facturación

Política de Privacidad de ExtensionPay: https://extensionpay.com/privacy

Importante: Los datos de pago son manejados completamente por ExtensionPay y Stripe. La Extensión nunca recibe ni almacena información de tu tarjeta de crédito.

InboxSDK (Integración con Gmail)

La Extensión usa InboxSDK (https://www.inboxsdk.com/) para integrarse con la interfaz de Gmail. InboxSDK es una librería que ayuda a la Extensión a interactuar con la UI de Gmail para mostrar la barra lateral y detectar contactos de email. InboxSDK no recopila, transmite ni almacena tus datos de email.

Compartir Datos con Terceros

NO compartimos tus datos con terceros para publicidad, análisis ni intermediación de datos.

La Extensión solo se comunica con:

  • Tu Instancia de Odoo: La Extensión envía peticiones API a la URL de instancia de Odoo que configuras para recuperar y crear registros
  • ExtensionPay: Solo para gestión de suscripciones (email, estado de suscripción, procesamiento de pagos)

La Extensión NO:

  • Vende tus datos a intermediarios de datos
  • Comparte datos con fines publicitarios
  • Transmite datos a servicios de análisis o seguimiento
  • Envía datos de Gmail u Odoo a los servidores del desarrollador de la extensión

Política de Datos de Usuario de Servicios de API de Google

El uso y transferencia de información recibida de las APIs de Google por Integración de Odoo para Gmail cumple con Google API Services User Data Policy, incluyendo los requisitos de Uso Limitado.

Cumplimiento de Uso Limitado:

  • La Extensión solo accede a datos de Gmail cuando abres explícitamente un email
  • Los datos de email se usan exclusivamente para el propósito declarado: integración con tu CRM de Odoo
  • Ningún dato de Gmail se usa para mostrar anuncios
  • Los datos de Gmail no se transfieren a terceros (excepto tu propia instancia de Odoo)
  • Los datos de Gmail no se usan para determinar solvencia crediticia ni propósitos de préstamos

Permisos Explicados

La Extensión solicita los siguientes permisos de Chrome:

Permisos Requeridos (Otorgados en la Instalación)

  • scripting: Requerido para inyectar la interfaz de la barra lateral en Gmail
  • storage: Requerido para guardar tu configuración de Odoo y almacenar respuestas de API localmente
  • host_permissions (mail.google.com): Requerido para acceder a la interfaz de Gmail y mostrar la barra lateral
  • host_permissions (*.odoo.com): Requerido para comunicarse con instancias cloud de Odoo Online (ej: tuempresa.odoo.com)
  • host_permissions (extensionpay.com): Requerido para gestión de suscripciones y procesamiento de pagos

Permisos Opcionales (Solicitados Cuando Se Necesitan)

La Extensión solicita permisos opcionales para instancias de Odoo personalizadas:

  • optional_host_permissions (https://*/*): Usado SOLO si te conectas a una instancia de Odoo auto-alojada en un dominio personalizado (ej: odoo.tuempresa.com o erp.ejemplo.com). Este permiso solo se solicita cuando configuras explícitamente una URL de Odoo personalizada.

Nota: La Extensión solo soporta conexiones HTTPS por seguridad. Las conexiones HTTP no están permitidas.

Cuándo Se Solicitan los Permisos Opcionales

Chrome te pedirá que otorgues permisos opcionales cuando:

  • Configures una URL de Odoo personalizada que no sea un dominio *.odoo.com
  • Elijas explícitamente conectarte a una instancia de Odoo auto-alojada

Puedes denegar estos permisos si solo usas Odoo Online (*.odoo.com). La Extensión funcionará normalmente con Odoo Online sin otorgar permisos opcionales.

¿Por Qué Estos Permisos Opcionales?

Muchas organizaciones alojan Odoo en sus propios servidores con dominios personalizados (ej: erp.acme.com, odoo.miempresa.net). Para dar soporte a estos usuarios, la Extensión solicita permisos opcionales amplios. Sin embargo:

  • Estos permisos SOLO se usan para comunicarse con TU instancia de Odoo configurada
  • NUNCA se usan para acceder a otros sitios web
  • Puedes revocarlos en cualquier momento en la configuración de Chrome
  • La Extensión solo los solicitará si introduces una URL de Odoo personalizada

Tus Derechos sobre los Datos

Tienes los siguientes derechos respecto a tus datos:

  • Acceso: Todos tus datos se almacenan localmente y en tu instancia de Odoo, que tú controlas
  • Eliminación: Puedes borrar todos los datos de la Extensión:
    • Haciendo clic en el botón de actualizar (borra la caché)
    • Eliminando la Extensión de Chrome (borra todos los datos locales)
  • Portabilidad: Tus datos permanecen en tu instancia de Odoo y son accesibles mediante las funciones de exportación de Odoo
  • Control: Puedes revocar el acceso de la Extensión en cualquier momento eliminándola

Retención de Datos

La Extensión retiene datos de la siguiente manera:

  • Datos de Configuración: Almacenados hasta que elimines la Extensión
  • Respuestas de API en Caché: Eliminadas automáticamente después de 5 minutos
  • Preferencias de Usuario: Almacenadas hasta que elimines la Extensión

Cuando desinstalas la Extensión, todos los datos almacenados localmente son eliminados automáticamente por Chrome.

Privacidad de Menores

La Extensión no está destinada para uso por menores de 13 años. No recopilamos intencionadamente información personal de menores de 13 años.

Cambios en Esta Política de Privacidad

Podemos actualizar esta política de privacidad de vez en cuando. Notificaremos a los usuarios de cualquier cambio material actualizando la fecha de "Última actualización" en la parte superior de esta política y, si es necesario, mediante las notificaciones de actualización de la Extensión.

El uso continuado de la Extensión después de los cambios constituye la aceptación de la política de privacidad actualizada.
