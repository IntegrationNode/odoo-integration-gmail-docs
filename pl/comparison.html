Nasze rozszerzenie

Rozszerzenie Chrome zbudowane z natywnymi API dla maksymalnej wydajności i stabilności.

Kluczowe zalety

  • ✓ 10-20x szybszy czas ładowania (100-300ms)
  • ✓ Stabilne połączenie, bez rozłączeń
  • ✓ Wyszukiwanie w czasie rzeczywistym we wszystkich rekordach
  • ✓ Działa na każdej instancji Odoo bez zmian w kodzie
  • ✓ Ukryj sekcje, aby zmniejszyć obciążenie danych
  • ✓ Ulepszone logowanie e-maili z załącznikami
  • ✓ Grupowanie rekordów według etapu i przeglądanie ostatnich aktywności

Odoo Gmail Plugin

Oficjalny dodatek zbudowany na Google Apps Script.

Funkcje

  • ✓ Przeglądaj kontakty i informacje o firmach
  • ✓ Twórz leady i loguj e-maile
  • ✓ Oficjalne wsparcie Odoo
  • ✗ Wolniejszy czas ładowania (2-6 sekund)
  • ✗ Częste rozłączenia
  • ✗ Wymaga zmian w kodzie dla samodzielnie hostowanych
