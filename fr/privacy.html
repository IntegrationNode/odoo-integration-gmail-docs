Politique de confidentialité

Dernière mise à jour: January 16, 2026

Aperçu

Intégration Odoo pour Gmail ("l'Extension") s'engage à protéger votre vie privée. Cette politique de confidentialité explique comment l'Extension collecte, utilise et protège vos informations.

Développeur : OdooIntegrations Team

Informations que nous collectons

Données accessibles depuis Gmail

L'Extension accède aux informations suivantes de Gmail lorsque vous ouvrez explicitement un email :

  • Adresses email expéditeur et destinataire : Utilisées pour rechercher les contacts correspondants dans votre instance Odoo
  • Sujet et corps de l'email : Utilisés lorsque vous choisissez de créer des opportunités, tâches ou tickets dans Odoo
  • Date et métadonnées de l'email : Utilisées pour enregistrer les emails dans les fiches Odoo

Données accessibles depuis Odoo

L'Extension récupère les informations suivantes de votre instance Odoo :

  • Informations de contact (noms, emails, téléphones, entreprises)
  • Opportunités CRM associées aux contacts
  • Tâches projet associées aux contacts
  • Tickets helpdesk associés aux contacts
  • Informations entreprise et contacts associés

Données de configuration

L'Extension stocke la configuration de votre instance Odoo localement :

  • URL de l'instance Odoo
  • Nom de la base de données
  • Identifiants d'authentification
  • Préférences utilisateur (paramètres des sections visibles)

Comment nous utilisons vos informations

L'Extension utilise vos informations uniquement aux fins suivantes :

  • Afficher les informations de contact : Montrer les détails des contacts Odoo pertinents dans la barre latérale de Gmail
  • Créer des enregistrements Odoo : Lorsque vous cliquez sur les boutons "Créer", le contenu de l'email est envoyé à votre instance Odoo pour créer des opportunités, tâches ou tickets
  • Enregistrer les emails : Lorsque vous cliquez sur les boutons "Enregistrer l'email", le contenu est sauvegardé dans le chatter de votre instance Odoo
  • Rechercher et filtrer : Rechercher parmi les opportunités, tâches et tickets associés aux contacts
  • Mise en cache : Mettre temporairement en cache les réponses API Odoo localement pour améliorer les performances

Stockage et sécurité des données

Stockage local uniquement : Toutes les données sont stockées localement sur votre appareil via les APIs de stockage Chrome. Aucune donnée n'est transmise à des serveurs tiers, sauf à votre propre instance Odoo.

Transmission des données : L'Extension communique directement avec VOTRE instance Odoo via HTTPS. Vos identifiants Odoo et données ne sont jamais envoyés au développeur de l'extension ni à des tiers.

Gestion du cache : Les données en cache sont automatiquement effacées après 5 minutes ou lorsque vous cliquez sur le bouton actualiser. Vous pouvez également effacer toutes les données en supprimant l'extension.

Services tiers

LemonSqueezy (Traitement des paiements)

L'Extension utilise LemonSqueezy (https://www.lemonsqueezy.com/) pour la gestion des abonnements et le traitement des paiements. Lorsque vous souscrivez au plan PLUS, LemonSqueezy collecte :

  • Votre adresse email pour la gestion du compte et les reçus
  • Informations de paiement (traitées de manière sécurisée, non stockées par l'Extension)
  • Statut d'abonnement et informations de facturation

Politique de confidentialité LemonSqueezy : https://www.lemonsqueezy.com/privacy

Important : Les données de paiement sont entièrement gérées par LemonSqueezy. L'Extension ne reçoit ni ne stocke jamais vos informations de carte bancaire.

InboxSDK (Intégration Gmail)

L'Extension utilise InboxSDK (https://www.inboxsdk.com/) pour s'intégrer à l'interface Gmail. InboxSDK est une bibliothèque qui aide l'Extension à interagir avec l'interface Gmail pour afficher la barre latérale et détecter les contacts email. InboxSDK ne collecte, transmet ni stocke vos données email.

Partage de données avec des tiers

Nous NE partageons PAS vos données avec des tiers à des fins publicitaires, analytiques ou de courtage de données.

L'Extension communique uniquement avec :

  • Votre instance Odoo : L'Extension envoie des requêtes API à l'URL Odoo que vous configurez pour récupérer et créer des enregistrements
  • LemonSqueezy: Uniquement pour la gestion des abonnements (email, statut d'abonnement, traitement des paiements)
  • Google Analytics: Statistiques d'utilisation anonymes (utilisation des fonctionnalités, erreurs). Aucune PII ni contenu email/Odoo.

L'Extension NE fait PAS :

  • Vendre vos données à des courtiers de données
  • Partager des données à des fins publicitaires
  • Envoyer des données Gmail ou Odoo aux serveurs du développeur de l'extension

Politique d'utilisation des données des services API Google

L'utilisation et le transfert par Intégration Odoo pour Gmail des informations reçues des APIs Google sont conformes à Google API Services User Data Policy, y compris les exigences d'utilisation limitée.

Conformité à l'utilisation limitée :

  • L'Extension n'accède aux données Gmail que lorsque vous ouvrez explicitement un email
  • Les données email sont utilisées exclusivement pour l'objectif déclaré : l'intégration avec votre CRM Odoo
  • Aucune donnée Gmail n'est utilisée pour diffuser des publicités
  • Les données Gmail ne sont pas transférées à des tiers (sauf à votre propre instance Odoo)
  • Les données Gmail ne sont pas utilisées pour déterminer la solvabilité ou à des fins de prêt

Explication des autorisations

L'Extension demande les autorisations Chrome suivantes :

Autorisations requises (Accordées à l'installation)

  • scripting: Nécessaire pour injecter l'interface de la barre latérale dans Gmail
  • storage: Nécessaire pour sauvegarder votre configuration Odoo et mettre en cache les réponses API localement
  • host_permissions (mail.google.com): Nécessaire pour accéder à l'interface Gmail et afficher la barre latérale
  • host_permissions (*.odoo.com): Nécessaire pour communiquer avec les instances cloud Odoo Online (ex: votreentreprise.odoo.com)
  • host_permissions (lemonsqueezy.com): Nécessaire pour la gestion des abonnements et le traitement des paiements

Autorisations optionnelles (Demandées si nécessaire)

L'Extension demande des autorisations optionnelles pour les instances Odoo personnalisées :

  • optional_host_permissions (https://*/*): Utilisé UNIQUEMENT si vous vous connectez à une instance Odoo auto-hébergée sur un domaine personnalisé (ex: odoo.votreentreprise.com ou erp.exemple.com). Cette autorisation n'est demandée que lorsque vous configurez explicitement une URL Odoo personnalisée.

Note : L'Extension ne supporte que les connexions HTTPS pour la sécurité. Les connexions HTTP ne sont pas autorisées.

Quand les autorisations optionnelles sont demandées

Chrome vous demandera d'accorder des autorisations optionnelles lorsque :

  • Vous configurez une URL Odoo personnalisée qui n'est pas un domaine *.odoo.com
  • Vous choisissez explicitement de vous connecter à une instance Odoo auto-hébergée

Vous pouvez refuser ces autorisations si vous utilisez uniquement Odoo Online (*.odoo.com). L'Extension fonctionnera normalement avec Odoo Online sans accorder les autorisations optionnelles.

Pourquoi ces autorisations optionnelles ?

De nombreuses organisations hébergent Odoo sur leurs propres serveurs avec des domaines personnalisés (ex: erp.acme.com, odoo.monentreprise.net). Pour supporter ces utilisateurs, l'Extension demande des autorisations optionnelles larges. Cependant :

  • Ces autorisations sont UNIQUEMENT utilisées pour communiquer avec VOTRE instance Odoo configurée
  • Elles ne sont JAMAIS utilisées pour accéder à d'autres sites web
  • Vous pouvez les révoquer à tout moment dans les paramètres Chrome
  • L'Extension ne les demandera que si vous entrez une URL Odoo personnalisée

Vos droits sur vos données

Vous avez les droits suivants concernant vos données :

  • Accès : Toutes vos données sont stockées localement et dans votre instance Odoo, que vous contrôlez
  • Suppression : Vous pouvez effacer toutes les données de l'Extension en :
    • Cliquant sur le bouton actualiser (efface le cache)
    • Supprimant l'Extension de Chrome (efface toutes les données locales)
  • Portabilité : Vos données restent dans votre instance Odoo et sont accessibles via les fonctionnalités d'export d'Odoo
  • Contrôle : Vous pouvez révoquer l'accès de l'Extension à tout moment en la supprimant

Conservation des données

L'Extension conserve les données comme suit :

  • Données de configuration : Stockées jusqu'à ce que vous supprimiez l'Extension
  • Réponses API en cache : Automatiquement supprimées après 5 minutes
  • Préférences utilisateur : Stockées jusqu'à ce que vous supprimiez l'Extension

Lorsque vous désinstallez l'Extension, toutes les données stockées localement sont automatiquement supprimées par Chrome.

Confidentialité des enfants

L'Extension n'est pas destinée aux enfants de moins de 13 ans. Nous ne collectons pas sciemment d'informations personnelles d'enfants de moins de 13 ans.

Modifications de cette politique de confidentialité

Nous pouvons mettre à jour cette politique de confidentialité de temps en temps. Nous informerons les utilisateurs de tout changement important en mettant à jour la date "Dernière mise à jour" en haut de cette politique et, si nécessaire, via les notifications de mise à jour de l'Extension.

L'utilisation continue de l'Extension après les modifications constitue l'acceptation de la politique de confidentialité mise à jour.
