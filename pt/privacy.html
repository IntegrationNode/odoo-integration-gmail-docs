Política de Privacidade

Última atualização: January 25, 2026

Visão Geral

Integração Odoo para Gmail ("a Extensão") está comprometida em proteger sua privacidade. Esta política de privacidade explica como a Extensão coleta, usa e protege suas informações.

Desenvolvedor: OdooIntegrations Team

Informações que Coletamos

Dados Acessados do Gmail

A Extensão acessa as seguintes informações do Gmail quando você abre explicitamente um e-mail:

  • Endereços de e-mail do remetente e destinatário: Usados para pesquisar contatos correspondentes na sua instância Odoo
  • Assunto e corpo do e-mail: Usados quando você escolhe criar oportunidades, tarefas ou tickets no Odoo
  • Data e metadados do e-mail: Usados para registrar e-mails em registros Odoo

Dados Acessados do Odoo

A Extensão recupera as seguintes informações da sua instância Odoo:

  • Informações de contato (nomes, e-mails, telefones, empresas)
  • Oportunidades CRM associadas a contatos
  • Tarefas de projeto associadas a contatos
  • Tickets do helpdesk associados a contatos
  • Informações da empresa e contatos relacionados

Dados de Configuração

A Extensão armazena a configuração da sua instância Odoo localmente:

  • URL da instância Odoo
  • Nome do banco de dados
  • Credenciais de autenticação
  • Preferências do usuário (configurações de seções visíveis)

Como Usamos Suas Informações

A Extensão usa suas informações exclusivamente para os seguintes propósitos:

  • Exibir Informações de Contato: Mostrar detalhes relevantes de contatos Odoo na barra lateral do Gmail
  • Criar Registros Odoo: Quando você clica nos botões "Criar", o conteúdo do e-mail é enviado para sua instância Odoo para criar oportunidades, tarefas ou tickets
  • Registrar E-mails: Quando você clica nos botões "Registrar E-mail", o conteúdo do e-mail é salvo no chatter da sua instância Odoo
  • Pesquisar e Filtrar: Pesquisar em oportunidades, tarefas e tickets associados a contatos
  • Cache: Armazenar temporariamente respostas da API Odoo localmente para melhorar o desempenho

Armazenamento e Segurança de Dados

Apenas Armazenamento Local: Todos os dados são armazenados localmente no seu dispositivo usando as APIs de armazenamento do Chrome. Nenhum dado é transmitido para servidores de terceiros, exceto sua própria instância Odoo.

Transmissão de Dados: A Extensão se comunica diretamente com SUA instância Odoo usando HTTPS. Suas credenciais Odoo e dados nunca são enviados ao desenvolvedor da extensão ou a terceiros.

Gerenciamento de Cache: Dados em cache são automaticamente limpos após 5 minutos ou quando você clica no botão atualizar. Você também pode limpar todos os dados em cache removendo a extensão.

Serviços de Terceiros

LemonSqueezy (Processamento de Pagamentos)

A Extensão usa LemonSqueezy (https://www.lemonsqueezy.com/) para gerenciamento de assinaturas e processamento de pagamentos. Quando você assina o plano PLUS, o LemonSqueezy coleta:

  • Seu endereço de e-mail para gerenciamento de conta e recibos
  • Informações de pagamento (processadas de forma segura, não armazenadas pela Extensão)
  • Status da assinatura e informações de faturamento

Política de Privacidade do LemonSqueezy: https://www.lemonsqueezy.com/privacy

Importante: Os dados de pagamento são tratados inteiramente pelo LemonSqueezy. A Extensão nunca recebe ou armazena as informações do seu cartão de crédito.

InboxSDK (Integração Gmail)

A Extensão usa InboxSDK (https://www.inboxsdk.com/) para integrar com a interface do Gmail. O InboxSDK é uma biblioteca que ajuda a Extensão a interagir com a interface do Gmail para exibir a barra lateral e detectar contatos de e-mail. O InboxSDK não coleta, transmite ou armazena seus dados de e-mail.

PostHog (Análises)

A Extensão usa PostHog (https://posthog.com/) para análises de uso anônimas. O PostHog nos ajuda a entender como a extensão é usada e identificar problemas. Os dados coletados incluem:

  • ID de usuário anônimo (UUID aleatório, não vinculado à sua identidade)
  • Eventos de uso de funcionalidades (ex: "e-mail registrado", "registro criado")
  • Relatórios de erros para depuração (sem dados pessoais ou conteúdo de e-mail)

Política de Privacidade do PostHog: https://posthog.com/privacy

Desativar: Você pode desativar as análises a qualquer momento em Configurações > Conta > Privacidade.

Compartilhamento de Dados com Terceiros

NÃO compartilhamos seus dados com terceiros para fins de publicidade, análise ou corretagem de dados.

A Extensão só se comunica com:

  • Sua Instância Odoo: A Extensão envia solicitações de API para a URL da instância Odoo que você configura para recuperar e criar registros
  • LemonSqueezy: Apenas para gerenciamento de assinaturas (e-mail, status da assinatura, processamento de pagamentos)
  • PostHog: Estatísticas de uso anônimas (uso de recursos, erros). Sem PII ou conteúdo de e-mail/Odoo. Você pode desativar em Configurações > Conta > Privacidade.

A Extensão NÃO:

  • Vende seus dados para corretores de dados
  • Compartilha dados para fins publicitários
  • Envia dados do Gmail ou Odoo para os servidores do desenvolvedor da extensão

Política de Dados do Usuário dos Serviços de API do Google

O uso e transferência de informações recebidas das APIs do Google pela Integração Odoo para Gmail adere à Google API Services User Data Policy, incluindo os requisitos de Uso Limitado.

Conformidade com Uso Limitado:

  • A Extensão só acessa dados do Gmail quando você abre explicitamente um e-mail
  • Os dados de e-mail são usados exclusivamente para o propósito declarado: integração com seu CRM Odoo
  • Nenhum dado do Gmail é usado para exibir anúncios
  • Os dados do Gmail não são transferidos para terceiros (exceto sua própria instância Odoo)
  • Os dados do Gmail não são usados para determinar credibilidade ou fins de empréstimo

Permissões Explicadas

A Extensão solicita as seguintes permissões do Chrome:

Permissões Obrigatórias (Concedidas na Instalação)

  • scripting: Necessário para injetar a interface da barra lateral no Gmail
  • storage: Necessário para salvar sua configuração Odoo e armazenar respostas de API localmente
  • host_permissions (mail.google.com): Necessário para acessar a interface do Gmail e exibir a barra lateral
  • host_permissions (*.odoo.com): Necessário para comunicar com instâncias cloud Odoo Online (ex: suaempresa.odoo.com)
  • host_permissions (lemonsqueezy.com): Necessário para gerenciamento de assinaturas e processamento de pagamentos

Permissões Opcionais (Solicitadas Quando Necessário)

A Extensão solicita permissões opcionais para instâncias Odoo personalizadas:

  • optional_host_permissions (https://*/*): Usado APENAS se você conectar a uma instância Odoo auto-hospedada em um domínio personalizado (ex: odoo.suaempresa.com ou erp.exemplo.com). Esta permissão só é solicitada quando você configura explicitamente uma URL Odoo personalizada.

Nota: A Extensão só suporta conexões HTTPS para segurança. Conexões HTTP não são permitidas.

Quando as Permissões Opcionais São Solicitadas

O Chrome solicitará que você conceda permissões opcionais quando:

  • Você configurar uma URL Odoo personalizada que não é um domínio *.odoo.com
  • Você escolher explicitamente conectar a uma instância Odoo auto-hospedada

Você pode negar essas permissões se você só usa Odoo Online (*.odoo.com). A Extensão funcionará normalmente com Odoo Online sem conceder permissões opcionais.

Por Que Essas Permissões Opcionais?

Muitas organizações hospedam Odoo em seus próprios servidores com domínios personalizados (ex: erp.acme.com, odoo.mycompany.net). Para suportar esses usuários, a Extensão solicita permissões opcionais amplas. No entanto:

  • Essas permissões são usadas APENAS para comunicar com SUA instância Odoo configurada
  • Elas NUNCA são usadas para acessar outros sites
  • Você pode revogá-las a qualquer momento nas configurações do Chrome
  • A Extensão só solicitará se você inserir uma URL Odoo personalizada

Seus Direitos sobre os Dados

Você tem os seguintes direitos em relação aos seus dados:

  • Acesso: Todos os seus dados são armazenados localmente e na sua instância Odoo, que você controla
  • Exclusão: Você pode limpar todos os dados da Extensão:
    • Clicando no botão atualizar (limpa o cache)
    • Removendo a Extensão do Chrome (limpa todos os dados locais)
  • Portabilidade: Seus dados permanecem na sua instância Odoo e são acessíveis através dos recursos de exportação do Odoo
  • Controle: Você pode revogar o acesso da Extensão a qualquer momento removendo-a

Retenção de Dados

A Extensão retém dados da seguinte forma:

  • Dados de Configuração: Armazenados até você remover a Extensão
  • Respostas de API em Cache: Automaticamente excluídas após 5 minutos
  • Preferências do Usuário: Armazenadas até você remover a Extensão

Quando você desinstala a Extensão, todos os dados armazenados localmente são automaticamente removidos pelo Chrome.

Privacidade de Menores

A Extensão não é destinada ao uso por menores de 13 anos. Não coletamos conscientemente informações pessoais de menores de 13 anos.

Alterações nesta Política de Privacidade

Podemos atualizar esta política de privacidade periodicamente. Notificaremos os usuários sobre quaisquer alterações materiais atualizando a data "Última atualização" no topo desta política e, se necessário, através das notificações de atualização da Extensão.

O uso continuado da Extensão após as alterações constitui aceitação da política de privacidade atualizada.
