Extensão vs Plugin Gmail Odoo

Recomendada

Nossa Extensão

Extensão Chrome construída com APIs nativas para máximo desempenho e estabilidade.

Principais Vantagens

  • ✓ Tempos de carregamento 10-20x mais rápidos (100-300ms)
  • ✓ Conexão estável, sem desconexões
  • ✓ Pesquisa em tempo real em todos os registros
  • ✓ Funciona em qualquer instância Odoo sem alterações no código
  • ✓ Ocultar seções para reduzir carregamento de dados
  • ✓ Registro de e-mail aprimorado com anexos
  • ✓ Agrupar registros por etapa e ver atividades recentes

Plugin Gmail Odoo

Add-on oficial construído em Google Apps Script.

Recursos

  • ✓ Visualizar contatos e info da empresa
  • ✓ Criar leads e registrar e-mails
  • ✓ Suporte oficial Odoo
  • ✗ Tempos de carregamento mais lentos (2-6 segundos)
  • ✗ Desconexões frequentes
  • ✗ Requer alterações no código para auto-hospedado
